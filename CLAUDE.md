## Agent skills

### Issue tracker

Issues live as GitHub issues on the fork `cislunarspace/MinerU` (use `--repo` flag; there are multiple remotes). See `docs/agents/issue-tracker.md`.

### Triage labels

Five canonical roles: `needs-triage`, `needs-info`, `ready-for-agent`, `ready-for-human`, `wontfix`. See `docs/agents/triage-labels.md`.

### Domain docs

Single-context layout. `CONTEXT.md` at root, `docs/adr/` for architectural decisions. Both created lazily by `/grill-with-docs`. See `docs/agents/domain.md`.
